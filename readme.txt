   1. Upload Megumi Goroku folder to the wp-content/plugins directory in your WordPress installation. Or you can download and install it using the built in WordPress plugin installer.
   2. Activate the plugin in your Administration Panel.
   3. In the Oni Daiko Setting screen, you can change the title, the number of posts to show, show/hide postâ€™s content, the length of content to show. (Also you can changes the settings in each widgets settings.)

      `<?php echo oni_daiko_template_tag(); ?>`
